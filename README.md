# 🐟 ぎょぴちゃん判定器

画像分類の仕組みを、子ども向けに視覚的に学べる教育教材です。
ブラウザだけで動き、サーバーや学習は不要です（学習済みモデルを同梱）。

## 2つの「見かた」を切り替えられます

| モード | 見えるもの | 特徴 |
|--------|-----------|------|
| 👀 どこ見た？ | 注目マップ（オクルージョン） | 高精度。でも「どこ」を見たかだけ |
| 🎨 なぜ？ | 色・形の特徴と「効き具合」 | 理由が言葉になる。でも控えめ・間違えることも |

切り替えると、コンピュータの「あたまの中」（脳みそネットワーク図）の見た目も変わります。

- 👀 = 大きすぎて中がのぞけない脳（MobileNet）
- 🎨 = 中が読める小さな脳（色・形の特徴ネットワーク）

「正確だけど理由がわからない AI」と「理由はわかるけど間違える AI」を、同じ画像で見くらべられます。

## 使い方

1. 画像をドラッグ＆ドロップ
2. 「これ だれ？」を押す
3. 結果・自信メーター・注目マップ／判断カードを見る
4. モードを切り替えて見くらべる

## しくみ

- **モードA**: MobileNet（v1, alpha 0.25）の埋め込み + 転移学習した分類器。判定根拠は**オクルージョン**（画像の一部を隠して自信の変化を測る）で可視化
- **モードB**: 色・形の手作り特徴5つ（暖かさ・明るさ・鮮やかさ・輪郭・色の多さ）+ 小型ニューラルネット。各特徴の「ぎょぴっぽさ（一致度）」と「効き具合（寄与度・近似）」を表示
- すべてブラウザ内（TensorFlow.js）で完結。サーバー処理なし

## 学習画像について

学習に使った画像は**著作権のため公開していません**。
このリポジトリには**学習済みモデル（重みの数値のみ・画像は復元不可）**だけを同梱しています。

## ローカルで学習し直す（開発者向け）

`train.html` と `training_data/` が手元にある場合：

```bash
python -m http.server 8080
# ブラウザで http://localhost:8080/train.html を開き「学習」
# → http://localhost:8080/index.html で判定
```

学習結果はブラウザの IndexedDB に保存されます。
`export_model.mjs` で `model/` フォルダにファイル化できます。
